Feast (Feature Store) is a tool to manage storage and access of machine learning features.

Feast allows you to:

  • Ingest features coming from batch and streaming source
  • Retrieve the features for both training and (low-latency) inference
  • Easily discover available features

This Python library allows you to register features and entities in Feast, ingest feature values, and retrieve the values for model training and serving.
